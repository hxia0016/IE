What can this game do?

This game helps you to learn about recycling in an interactive way by placing the item to the right bin. Recommended for teaching children about recycling in a fun way.

How to play the game?

  1. Enable/Allow camera access
  2. Use your hand to pick the displayed item
  3. Move the item to the right recycling bin color

Game Rules

  1. Put garbage into correct the garbage bin.
  2. Red lid bins is for landfill.
  3. Yellow lid bins is for recycleing.
  4. Green lid bins is for organic garbage.
  5. Wrong classification will make fail in this game.
